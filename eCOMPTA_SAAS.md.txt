## 0. CONTEXTE MÉTIER

Tu codes une **plateforme SaaS multitenant** pour cabinets comptables au Bénin (zone UEMOA/OHADA). Elle automatise la saisie des factures via IA (OCR + LLM), génère des écritures conformes au **plan comptable SYSCOHADA Révisé**, et gère les abonnements des cabinets partenaires.

### Flux principal
```
Cabinet → Upload PDF → OCR Mistral AI → Classification GPT-4o-mini
→ Extraction champs → Revue comptable humaine → Validation
→ Génération écritures SYSCOHADA → Export GSheets / XLSX
```

### Acteurs
| Rôle | Description |
|------|-------------|
| `super_admin` | DRAWIS — gère toute la plateforme, tous les tenants |
| `admin` | Admin du cabinet — gère son tenant (users, config, abonnement) |
| `comptable` | Saisit, corrige, valide les factures |
| `auditeur` | Lecture seule — consulte journaux et historique |

## 1. STACK TECHNIQUE OBLIGATOIRE

```
Backend  : PHP 8.3 + Laravel 11
Frontend : Blade + Alpine.js + Tailwind CSS 3 (pas de React)
BDD      : PostgreSQL 16
Cache    : Redis 7
Queue    : Laravel Horizon (Redis)
Stockage : MinIO (S3-compatible, self-hosted)
Auth     : Laravel Sanctum (JWT-like tokens)
Docker   : Docker Compose (dev + prod)
CI/CD    : GitHub Actions
```

### Pourquoi ce stack
- Laravel = parfait pour multitenant avec Spatie Multitenancy
- Blade + Alpine.js = léger, pas besoin de SPA pour cette app
- PostgreSQL = Row Level Security natif pour isolation tenant
- MinIO = S3 sans coût externe, stockage PDF isolé par tenant
- Horizon = monitoring des jobs IA en temps réel

```dockerfile
    opcache sockets

# Redis extension
RUN pecl install redis && docker-php-ext-enable redis

# Composer
COPY --from=composer:2.7 /usr/bin/composer /usr/bin/composer

# User non-root
RUN usermod -u 1000 www-data

WORKDIR /var/www

COPY --chown=www-data:www-data . .

RUN composer install --no-dev --optimize-autoloader \
    && npm ci && npm run build \
    && chown -R www-data:www-data /var/www/storage /var/www/bootstrap/cache
```

## 5. MODÈLES ELOQUENT

### `app/Models/Tenant.php`
```php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Concerns\HasUuids;
use Illuminate\Database\Eloquent\SoftDeletes;
use Illuminate\Database\Eloquent\Relations\HasMany;
use Illuminate\Database\Eloquent\Relations\BelongsTo;

class Tenant extends Model
{
    use HasUuids, SoftDeletes;

    protected $fillable = [
        'nom', 'slug', 'ifu', 'email_contact', 'telephone',
        'adresse', 'ville', 'pays', 'devise', 'plan', 'statut',
        'quota_factures_mensuel', 'quota_users',
        'abonnement_expire_le', 'config_google_sheets', 'config_n8n', 'actif',
    ];

    protected $casts = [
        'config_google_sheets' => 'array',
        'config_n8n' => 'array',
        'actif' => 'boolean',
        'abonnement_expire_le' => 'datetime',
    ];

    public function users(): HasMany
    {
        return $this->hasMany(User::class);
    }

    public function factures(): HasMany
    {
        return $this->hasMany(Facture::class);
    }

    public function abonnements(): HasMany
    {
        return $this->hasMany(Abonnement::class);
    }

    public function abonnementActif(): ?Abonnement
    {
        return $this->abonnements()
            ->where('statut', 'actif')
            ->where('expire_le', '>', now())
            ->latest()
            ->first();
    }

    public function estActif(): bool
    {
        return $this->statut === 'actif' || $this->statut === 'trial';
    }

    public function facturesCeMois(): int
    {
        return $this->factures()
            ->whereMonth('created_at', now()->month)
            ->whereYear('created_at', now()->year)
            ->whereNotIn('statut', ['rejete', 'erreur'])
            ->count();
    }

    public function quotaDisponible(): bool
    {
        return $this->facturesCeMois() < $this->quota_factures_mensuel;
    }
}
```

```php
<?php

namespace App\Services;

use App\Models\Facture;
use App\Models\TraitementLog;
use Illuminate\Support\Facades\Http;
use Illuminate\Support\Facades\Log;

class N8nService
{
    private string $webhookUrl;
    private string $secret;
    private string $callbackUrl;

    public function __construct()
    {
        $this->webhookUrl  = config('services.n8n.webhook_url');
        $this->secret      = config('services.n8n.secret');
        $this->callbackUrl = config('services.n8n.callback_url');
    }

    /**
     * Déclenche le pipeline n8n pour traiter une facture PDF.
     * n8n est appelé via réseau interne Docker uniquement.
     */
    public function declencherTraitement(Facture $facture): bool
    {
        $payload = [
            'facture_id'   => $facture->id,
            'tenant_id'    => $facture->tenant_id,
            'pdf_url'      => $this->genererUrlSigneePdf($facture->pdf_path),
            'callback_url' => $this->callbackUrl . '/webhooks/n8n/callback',
            'timestamp'    => now()->toISOString(),
        ];

        // Signature HMAC pour authentifier l'appel
        $signature = hash_hmac('sha256', json_encode($payload), $this->secret);

        try {
            $response = Http::timeout(30)
                ->withHeaders([
                    'X-N8N-Secret'    => $signature,
                    'X-Tenant-Id'     => $facture->tenant_id,
                    'Content-Type'    => 'application/json',
                ])
                ->post($this->webhookUrl, $payload);

            if ($response->successful()) {
                $facture->update([
                    'statut'          => 'ocr_en_cours',
                    'n8n_started_at'  => now(),
                    'n8n_execution_id' => $response->json('executionId'),
                ]);

                TraitementLog::create([
                    'facture_id' => $facture->id,
                    'etape'      => 'ocr',
                    'statut'     => 'en_cours',
                    'message'    => 'Pipeline n8n déclenché avec succès',
                ]);

                return true;
            }

            Log::error('N8n webhook error', [
                'facture_id' => $facture->id,
                'status'     => $response->status(),
                'body'       => $response->body(),
            ]);

            return false;

        } catch (\Exception $e) {
            Log::error('N8n connection failed', [
                'facture_id' => $facture->id,
                'error'      => $e->getMessage(),
            ]);

            $facture->update(['statut' => 'erreur']);
            return false;
        }
    }

    /**
     * Vérifie la signature HMAC du callback n8n entrant.
     */
    public function verifierSignature(string $payload, string $signature): bool
    {
        $expected = hash_hmac('sha256', $payload, $this->secret);
        return hash_equals($expected, $signature);
    }

    private function genererUrlSigneePdf(string $path): string
    {
        return app(MiniOService::class)->urlSignee($path, 900); // 15 min
    }
}
```

```php
namespace App\Services;

use App\Models\Facture;

class SYSCOHADAService

    /**
     * Génère les écritures comptables SYSCOHADA à partir des données n8n.
     * Règles fiscales Bénin : AIB 1%, RIRF, Régimes B et D.
     */
    public function genererEcritures(Facture $facture, array $ecrituresN8n): void
    {
        // Supprimer les anciennes écritures si re-génération
        $facture->ecritures()->delete();

        $ordre = 0;

        foreach ($ecrituresN8n as $ligne) {
            EcritureComptable::create([
                'tenant_id'              => $facture->tenant_id,
                'facture_id'             => $facture->id,
                'journal'                => $this->determinerJournal($facture->type_document),
                'date_ecriture'          => $facture->date_facture ?? now()->toDateString(),
                'numero_piece'           => $facture->numero_facture,
                'numero_compte'          => $ligne['compte'],
                'libelle_compte'         => $ligne['libelle_compte'] ?? null,
                'libelle_ecriture'       => $ligne['libelle'],
                'debit'                  => $ligne['debit'] ?? 0,
                'credit'                 => $ligne['credit'] ?? 0,
                'devise'                 => 'XOF',
                'est_ecriture_reglement' => $ligne['est_reglement'] ?? false,
                'ordre_ligne'            => $ordre++,
            ]);
        }
    }

    /**
     * Génère les écritures manuellement (fallback si n8n ne fournit pas les écritures).
     * Couvre les cas VENTE, ACHAT, CHARGE avec fiscalité Bénin.
     */
    public function genererEcrituresManuel(Facture $facture): array
    {
        $donnees = array_merge(
            $facture->donnees_extraites ?? [],
            $facture->donnees_corrigees ?? []
        );

        $ecritures = match($facture->type_document) {
            'VENTE'  => $this->ecrituresVente($facture, $donnees),
            'ACHAT'  => $this->ecrituresAchat($facture, $donnees),
            'CHARGE' => $this->ecrituresCharge($facture, $donnees),
            default  => [],
        };

        return $ecritures;
    }

    private function ecrituresVente(Facture $facture, array $d): array
    {
        $ecritures = [];
        $ht  = (float) ($facture->montant_ht ?? 0);
        $ttc = (float) ($facture->montant_ttc ?? 0);
        $aib = (float) ($facture->montant_aib ?? 0);
        $rirf = (float) ($facture->montant_rirf ?? 0);
        $regimeD = ($facture->regime_fiscal === 'D');

        // Débit : 411 Clients (montant TTC - retenues)
        $montantClient = $ttc - $aib - $rirf;
        $ecritures[] = $this->ligne('4111', 'Clients', $montantClient, 0);

        if ($aib > 0) {
            $ecritures[] = $this->ligne('4473', 'RIRF à récupérer', $aib, 0);
        }

        if ($rirf > 0) {
            $ecritures[] = $this->ligne('4473', 'RIRF à récupérer', $rirf, 0);
        }

        // Crédit : 701 Ventes de marchandises
        $ecritures[] = $this->ligne('7011', 'Ventes de marchandises', 0, $ht);

        // Crédit : 4431 TVA facturée (seulement si régime B)
        if (!$regimeD && $facture->montant_tva > 0) {
            $ecritures[] = $this->ligne('4431', 'T.V.A. facturée sur ventes', 0, $facture->montant_tva);
        }

        // Écriture de règlement
        $ecritures[] = $this->ecritureReglement($ttc, $facture->mode_paiement, true);

        return $ecritures;
    }

    private function ecrituresAchat(Facture $facture, array $d): array
    {
        $ecritures = [];
        $ht  = (float) ($facture->montant_ht ?? 0);
        $ttc = (float) ($facture->montant_ttc ?? 0);
        $aib = (float) ($facture->montant_aib ?? 0);

        // Débit : 601 Achats de marchandises
        $ecritures[] = $this->ligne('6011', 'Achats de marchandises', $ht, 0);

        // Débit : 4452 TVA récupérable sur achats
        if ($facture->montant_tva > 0) {
            $ecritures[] = $this->ligne('4452', 'T.V.A. récupérable sur achats', $facture->montant_tva, 0);
        }

        // Débit : AIB à récupérer
        if ($aib > 0) {
            $ecritures[] = $this->ligne('44593', 'AIB à récupérer 1%', $aib, 0);
        }

        // Crédit : 401 Fournisseurs
        $ecritures[] = $this->ligne('4011', 'Fournisseurs', 0, $ttc);

        // Écriture de règlement
        $ecritures[] = $this->ecritureReglement($ttc, $facture->mode_paiement, false);

        return $ecritures;
    }

    private function ecrituresCharge(Facture $facture, array $d): array
    {
        $ecritures = [];
        $ht  = (float) ($facture->montant_ht ?? 0);
        $ttc = (float) ($facture->montant_ttc ?? 0);

        // Déterminer le compte de charge selon la nature (envoyé par n8n)
        $compteCharge = $d['compte_syscohada'] ?? '6228'; // Locations par défaut

        }

        $ecritures[] = $this->ligne('4011', 'Fournisseurs', 0, $ttc);
        $ecritures[] = $this->ecritureReglement($ttc, $facture->mode_paiement, false);

        return $ecritures;
    }

    /**
     * Écriture de règlement selon mode de paiement et montant.
     * Règle Bénin : < 100 000 FCFA → caisse, ≥ 100 000 → banque, mobile money → 531
     */
    private function ecritureReglement(float $montant, ?string $mode, bool $estVente): array
    {

            default                     => ['5711', 'Caisse en monnaie nationale'],
        };

        return $this->ligne(
            $compte[0],
            $compte[1],
            $estVente ? $montant : 0,
            $estVente ? 0 : $montant,
            true
        );
    }
```

### `app/Http/Controllers/Webhook/N8nCallbackController.php`
```php
<?php

namespace App\Http\Controllers\Webhook;

use App\Http\Controllers\Controller;
use App\Models\Facture;
use App\Models\TraitementLog;
use App\Services\N8nService;
use App\Services\SYSCOHADAService;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Log;

class N8nCallbackController extends Controller
{
    public function __construct(
        private N8nService       $n8n,
        private SYSCOHADAService $syscohada
    ) {}

    /**
     * Point d'entrée des résultats du pipeline n8n.
     * Cette route est appelée par n8n uniquement (réseau interne).
     * Vérification HMAC obligatoire.
     */
    public function recevoir(Request $request)
    {
        // Vérification signature HMAC
        $signature = $request->header('X-N8N-Secret');
        $payload   = $request->getContent();

        if (!$this->n8n->verifierSignature($payload, $signature)) {
            Log::warning('N8n callback: signature invalide', [
                'ip' => $request->ip(),
            ]);
            return response()->json(['error' => 'Non autorisé'], 403);
        }

        $data      = $request->json()->all();
        $factureId = $data['facture_id'] ?? null;
        $etape     = $data['etape'] ?? null;

        if (!$factureId || !$etape) {
            return response()->json(['error' => 'Données manquantes'], 400);
        }

        $facture = Facture::find($factureId);
        if (!$facture) {
            return response()->json(['error' => 'Facture introuvable'], 404);
        }

        try {
            $this->traiterEtape($facture, $etape, $data);
            return response()->json(['ok' => true]);
        } catch (\Exception $e) {
            Log::error('N8n callback traitement erreur', [
                'facture_id' => $factureId,
                'etape'      => $etape,
                'error'      => $e->getMessage(),
            ]);

            $facture->update(['statut' => 'erreur']);

            return response()->json(['error' => $e->getMessage()], 500);
        }
    }

    private function traiterEtape(Facture $facture, string $etape, array $data): void
    {
        $dureeMs = $data['duree_ms'] ?? null;

        match($etape) {
            'ocr_termine' => $this->traiterOCR($facture, $data, $dureeMs),
            'classification_terminee' => $this->traiterClassification($facture, $data, $dureeMs),
            'extraction_terminee' => $this->traiterExtraction($facture, $data, $dureeMs),
            'ecritures_generees' => $this->traiterEcritures($facture, $data, $dureeMs),
            'erreur' => $this->traiterErreur($facture, $data),
            default => Log::warning("N8n callback: étape inconnue {$etape}"),
        };
    }

    private function traiterOCR(Facture $facture, array $data, ?int $dureeMs): void
    {
        $facture->update([
            'statut'        => 'ocr_termine',
            'ocr_texte'     => $data['ocr_texte'],
            'nombre_pages'  => $data['nombre_pages'] ?? null,
        ]);

        TraitementLog::create([
            'facture_id' => $facture->id,
            'etape'      => 'ocr',
            'statut'     => 'succes',
            'duree_ms'   => $dureeMs,
            'message'    => "OCR terminé — {$data['nombre_pages']} page(s) extraite(s)",
        ]);
    }

    private function traiterClassification(Facture $facture, array $data, ?int $dureeMs): void
    {
        $facture->update([
            'statut'                       => 'classification_terminee',
            'type_document'                => $data['type_document'],
            'sous_type'                    => $data['sous_type'] ?? null,
            'score_confiance_classification' => $data['score_confiance'] ?? null,
            'justification_ia'             => $data['justification'] ?? null,
        ]);

        TraitementLog::create([
            'facture_id' => $facture->id,
            'etape'      => 'classification',
            'statut'     => 'succes',
            'duree_ms'   => $dureeMs,
            'message'    => "Type: {$data['type_document']} (confiance: {$data['score_confiance']}%)",
        ]);
    }

    private function traiterExtraction(Facture $facture, array $data, ?int $dureeMs): void
    {
        $extraits = $data['donnees_extraites'] ?? [];

        $facture->update([
            'statut'            => 'a_valider',
            'donnees_extraites' => $extraits,
            'numero_facture'    => $extraits['numero_facture'] ?? null,
            'fournisseur_client' => $extraits['fournisseur'] ?? $extraits['client'] ?? null,
            'ifu_tiers'         => $extraits['ifu'] ?? null,
            'code_mecef'        => $extraits['code_mecef'] ?? null,
            'date_facture'      => $extraits['date_facture'] ?? null,
            'montant_ht'        => $extraits['montant_ht'] ?? null,
            'montant_tva'       => $extraits['montant_tva'] ?? null,
            'montant_ttc'       => $extraits['montant_ttc'] ?? null,
            'montant_aib'       => $extraits['montant_aib'] ?? null,
            'montant_rirf'      => $extraits['montant_rirf'] ?? null,
            'regime_fiscal'     => $extraits['regime'] ?? null,
            'mode_paiement'     => $extraits['mode_paiement'] ?? null,
        ]);

        TraitementLog::create([
            'facture_id' => $facture->id,
            'etape'      => 'extraction',
            'statut'     => 'succes',
            'duree_ms'   => $dureeMs,
            'message'    => 'Données extraites — en attente de validation comptable',
        ]);
    }

    private function traiterEcritures(Facture $facture, array $data, ?int $dureeMs): void
    {
        if (!empty($data['ecritures'])) {
            $this->syscohada->genererEcritures($facture, $data['ecritures']);
        }

        TraitementLog::create([
            'facture_id' => $facture->id,
            'etape'      => 'generation_ecritures',
            'statut'     => 'succes',
            'duree_ms'   => $dureeMs,
            'message'    => count($data['ecritures'] ?? []) . ' écritures générées',
        ]);
    }

    private function traiterErreur(Facture $facture, array $data): void
    {
        $facture->update(['statut' => 'erreur']);

        TraitementLog::create([
            'facture_id' => $facture->id,
            'etape'      => $data['etape_echouee'] ?? 'unknown',
            'statut'     => 'erreur',
            'message'    => $data['message_erreur'] ?? 'Erreur inconnue',
        ]);
    }
}
```

```php
<?php

namespace App\Http\Controllers\Webhook;

use App\Http\Controllers\Controller;
use App\Models\Abonnement;
use App\Models\Plan;
use App\Models\Tenant;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Log;

class FeexPayController extends Controller
{
    /**
     * Reçoit la confirmation de paiement FeexPay.
     * Activation ou renouvellement de l'abonnement du cabinet.
     */
    public function callback(Request $request)
    {
        $statut     = $request->input('status');
        $callbackInfo = json_decode($request->input('callback_info'), true);
        $montant    = (int) $request->input('amount');
        $transactionId = $request->input('id') ?? $request->input('transaction_id');

        if ($statut !== 'SUCCESSFUL') {
            Log::info('FeexPay paiement non réussi', ['statut' => $statut, 'data' => $request->all()]);
            return response()->json(['ok' => true]); // On ignore les échecs
        }

        $tenantId = $callbackInfo['tenant_id'] ?? null;
        $planSlug = $callbackInfo['plan'] ?? null;

        if (!$tenantId || !$planSlug) {
            Log::error('FeexPay callback: données manquantes', $request->all());
            return response()->json(['error' => 'Données manquantes'], 400);
        }

        $tenant = Tenant::find($tenantId);
        $plan   = Plan::where('slug', $planSlug)->first();

        if (!$tenant || !$plan) {
            Log::error('FeexPay callback: tenant ou plan introuvable', compact('tenantId', 'planSlug'));
            return response()->json(['error' => 'Tenant ou plan introuvable'], 404);
        }

        // Créer l'enregistrement d'abonnement
        Abonnement::create([
            'tenant_id'            => $tenant->id,
            'plan_id'              => $plan->id,
            'statut'               => 'actif',
            'processeur_paiement'  => 'feexpay',
            'transaction_id'       => $transactionId,
            'montant_xof'          => $montant,
            'debut_le'             => now(),
            'expire_le'            => now()->addDays(30),
            'metadata_paiement'    => $request->all(),
        ]);

        // Mettre à jour le tenant
        $tenant->update([
            'plan'                    => $plan->slug,
            'statut'                  => 'actif',
            'quota_factures_mensuel'  => $plan->quota_factures,
            'quota_users'             => $plan->quota_users,
            'abonnement_expire_le'    => now()->addDays(30),
        ]);

        Log::info("Abonnement activé — Tenant: {$tenant->nom} — Plan: {$plan->nom}");

        return response()->json(['ok' => true]);
    }
}
```

```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class CheckSubscription
{
    public function handle(Request $request, Closure $next)
    {
        $user = $request->user();

        // Super admin = pas de vérification
        if ($user->role === 'super_admin') {
            return $next($request);
        }

        $tenant = $user->tenant;

        if (!$tenant || !$tenant->estActif()) {
            if ($request->expectsJson()) {
                return response()->json([
                    'error'    => 'Abonnement expiré ou inactif',
                    'redirect' => route('abonnement.index'),
                ], 402);
            }

            return redirect()
                ->route('abonnement.index')
                ->withErrors(['abonnement' => 'Votre abonnement est expiré. Veuillez renouveler pour continuer.']);
        }

        return $next($request);
    }
}
```

```php
namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class CheckQuota
{
    public function handle(Request $request, Closure $next)
    {
        $tenant = $request->user()->tenant;

        if ($tenant && !$tenant->quotaDisponible()) {
            $utilisees = $tenant->facturesCeMois();
            $quota     = $tenant->quota_factures_mensuel;

            if ($request->expectsJson()) {
                return response()->json([
                    'error' => "Quota mensuel atteint ({$utilisees}/{$quota} factures)",
                ], 402);
            }

            return back()->withErrors([
                'quota' => "Quota mensuel atteint ({$utilisees}/{$quota}). Passez à un plan supérieur."
            ]);
        }

        return $next($request);
    }
}
```

```php
<?php

namespace App\Jobs;

use App\Models\Facture;
use App\Models\TraitementLog;
use App\Services\N8nService;
use Illuminate\Bus\Queueable;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Foundation\Bus\Dispatchable;
use Illuminate\Queue\InteractsWithQueue;
use Illuminate\Queue\SerializesModels;
use Illuminate\Support\Facades\Log;

class TraiterFacturePDF implements ShouldQueue
{
    use Dispatchable, InteractsWithQueue, Queueable, SerializesModels;

    public int $tries   = 3;
    public int $timeout = 300; // 5 minutes max

    public function __construct(private Facture $facture) {}

    public function handle(N8nService $n8n): void
    {
        Log::info("TraiterFacturePDF: début", ['facture_id' => $this->facture->id]);

        $succes = $n8n->declencherTraitement($this->facture);

        if (!$succes) {
            TraitementLog::create([
                'facture_id' => $this->facture->id,
                'etape'      => 'upload',
                'statut'     => 'erreur',
                'message'    => 'Impossible de contacter n8n — nouvelle tentative dans 1 minute',
            ]);

            $this->release(60); // Retry après 60 secondes
        }
    }

    public function failed(\Throwable $e): void
    {
        $this->facture->update(['statut' => 'erreur']);

        TraitementLog::create([
            'facture_id' => $this->facture->id,
            'etape'      => 'upload',
            'statut'     => 'erreur',
            'message'    => "Échec définitif: {$e->getMessage()}",
        ]);

        Log::error("TraiterFacturePDF: échec définitif", [
            'facture_id' => $this->facture->id,
            'error'      => $e->getMessage(),
        ]);
    }
}
```

## 15. COMMANDE CRON — EXPIRATION ABONNEMENTS

```php
<?php
// app/Console/Commands/ExpireSubscriptions.php

namespace App\Console\Commands;

use App\Models\Tenant;
use Illuminate\Console\Command;
use Illuminate\Support\Facades\Mail;

class ExpireSubscriptions extends Command
{
    protected $signature   = 'abonnements:expirer';
    protected $description = 'Expire les abonnements arrivés à échéance et envoie les rappels';

    public function handle(): void
    {
        // 1. Expirer les abonnements dépassés
        $expires = Tenant::where('statut', 'actif')
            ->where('abonnement_expire_le', '<', now())
            ->get();

        foreach ($expires as $tenant) {
            $tenant->update(['statut' => 'expire']);
            $this->info("Expiré: {$tenant->nom}");
        }

        $this->info("{$expires->count()} abonnement(s) expiré(s).");

        // 2. Rappels J-3
        $bientotExpires = Tenant::where('statut', 'actif')
            ->whereBetween('abonnement_expire_le', [now(), now()->addDays(3)])
            ->get();

        foreach ($bientotExpires as $tenant) {
            // Mail::to($tenant->email_contact)->send(new RappelAbonnementMail($tenant));
            $this->info("Rappel envoyé: {$tenant->nom}");
        }
    }
}
```

## 19. RÈGLES MÉTIER CRITIQUES À RESPECTER

```
RÈGLE 1 — Isolation tenant
  → CHAQUE requête SQL sur factures/ecritures/users DOIT filtrer par tenant_id
  → Ne JAMAIS faire Facture::all() sans ->where('tenant_id', ...)
  → Tester avec deux tenants en parallèle

RÈGLE 2 — Signature HMAC n8n
  → Tous les appels backend→n8n : Header X-N8N-Secret = hmac_sha256(payload, SECRET)
  → Tous les callbacks n8n→backend : vérifier la signature avant tout traitement
  → Refus 403 si signature invalide ou absente

RÈGLE 3 — URLs PDF signées uniquement
  → Jamais de URL publique permanente pour un PDF
  → Toujours générer une URL signée MinIO (15 min pour n8n, 1h pour UI)
  → Stocker uniquement le chemin relatif en BDD

RÈGLE 4 — Équilibre SYSCOHADA
  → TOUTE écriture doit être équilibrée : Σ débit = Σ crédit
  → Tester estEquilibree() avant validation
  → Bloquer la validation si déséquilibre

RÈGLE 5 — Fiscalité Bénin
  → Régime B (18% TVA) → compte 4431 obligatoire en vente, 4452 en achat
  → Régime D (exonéré)  → AUCUNE ligne TVA
  → AIB 1% → 4472 crédit vente / 44593 débit achat
  → RIRF → 4473 débit vente
  → Règlement : < 100k FCFA = caisse 5711, ≥ 100k = banque 5211, mobile = 5731

RÈGLE 6 — n8n invisible
  → Port n8n JAMAIS exposé dans docker-compose.yml
  → L'URL n8n JAMAIS retournée au frontend
  → Seul le backend connaît N8N_WEBHOOK_URL
```
